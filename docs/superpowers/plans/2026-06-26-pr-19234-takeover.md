### Task 2: Apply the two code fixes

**Files:**
- Modify: `server/handlers/meshery_pattern_handler.go` — lines ~758 and ~768

**The two problems in the PR diff:**
1. Validation block uses `ErrGetPattern` — should be `ErrDeletePattern` (function exists in `error.go:575`).
2. `eventBuilder` calls `uuid.FromStringOrNil(patternID)` again — `patternUUID` is already parsed above and should be reused.

- [ ] **Step 1: Fix `ErrGetPattern` → `ErrDeletePattern`**

In `server/handlers/meshery_pattern_handler.go`, find:
```go
err := ErrGetPattern(fmt.Errorf("invalid design ID: %s", patternID))
```
Replace with:
```go
err := ErrDeletePattern(fmt.Errorf("invalid design ID: %s", patternID))
```

- [ ] **Step 2: Fix redundant UUID parse in `eventBuilder`**

In the same file, find (the line immediately after the validation block's closing brace):
```go
eventBuilder := events.NewEvent().FromOwner(userID).FromSystem(*h.SystemID).WithCategory("pattern").WithAction("delete").ActedUpon(uuid.FromStringOrNil(patternID))
```
Replace with:
```go
eventBuilder := events.NewEvent().FromOwner(userID).FromSystem(*h.SystemID).WithCategory("pattern").WithAction("delete").ActedUpon(patternUUID)
```
