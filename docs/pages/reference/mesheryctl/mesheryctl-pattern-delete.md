# mesheryctl pattern delete

Delete pattern file

## Synopsis

delete pattern file will trigger deletion of the pattern file

<pre class='codeblock-pre'>
<div class='codeblock'>
mesheryctl pattern delete [flags]

</div>
</pre> 

## Examples

delete a pattern file
<pre class='codeblock-pre'>
<div class='codeblock'>
mesheryctl pattern delete [file | URL]

</div>
</pre>
