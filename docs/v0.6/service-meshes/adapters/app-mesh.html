Meshery Adapter for App Mesh

Cloud Native Infrastructure Adapter Status Adapter Version Earliest Supported Infrastructure Version
App Mesh beta v1.4.1

Lifecycle management

The Meshery Adapter for App Mesh can install v1.4.1 of App Mesh service mesh. A number of sample applications can be installed using the Meshery Adapter for App Mesh.

Features

  1. Lifecycle management of App Mesh
  2. Lifecycle management of sample applications
  3. Performance testing

Sample Applications

The Meshery Adapter for App Mesh includes a handful of sample applications. Use Meshery to deploy any of these sample applications.

  • Emojivoto

    • A microservice application that allows users to vote for their favorite emoji, and tracks votes received on a leaderboard.

  • Bookinfo

    • Follow this tutorial workshop to set up and deploy the BookInfo sample app on Istio using Meshery.

  • Httpbin

    • Httpbin is a simple HTTP request and response service.
