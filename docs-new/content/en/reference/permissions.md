Permissions are represented as <b>keys</b>, each serving as a unique identifier for a specific permission. One or more keys can be grouped together and assigned to a <b>keychain</b>.<br>

{{% alert color="info" title="Customizable Permissions" %}}Default permissions can be easily customized by simply creating your own keychains and roles.{{% /alert %}}

{{% alert color="info" title="Contributing to Permission Keys" %}}To contribute permission keys, see the <a href='/extensibility/authorization'>Authorization Extensibility Documentation</a>. This guide explains Meshery's extensible authorization system.{{% /alert %}}
